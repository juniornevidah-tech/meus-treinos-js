# 🔥 Meus Treinos de JavaScript

Este repositório foi criado para documentar a minha evolução em JavaScript antes de migrar para o React.js focando em desenvolvimento freelance.

## 🛠️ Projetos Inclusos:

*   **Carrinho de Compras:** Simulação de um checkout de e-commerce com cálculo de total, aplicação de cupom de desconto (estudo de escopo) e simulação de pagamento assíncrono com `async/await`.
*   **Buscador de CEP:** Consumo de API para busca de endereços.

## 🚀 Tecnologias Utilizadas:
*   JavaScript (ES6+)
*   Node.js (Execução local)
*   Git & GitHub
